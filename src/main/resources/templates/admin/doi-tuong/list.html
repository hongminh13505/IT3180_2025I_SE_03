Hướng dẫn Import

  • Tải file mẫu trước khi import để đảm bảo định dạng đúng
  • Điền đầy đủ các trường bắt buộc (đánh dấu *)
  • CCCD: Phải có đúng 12 ký tự số và không trùng lặp
  • Ngày sinh: Định dạng dd/MM/yyyy (ví dụ: 01/01/1990)
  • Giới tính: nam hoặc nu
  • Vai trò: ban_quan_tri, co_quan_chuc_nang, ke_toan, nguoi_dung_thuong, khong_dung_he_thong
  • Là cư dân: true hoặc false
